# nodepay

## usage

### first usage 
```
python3 -m venv ./.venv
source ./.venv/bin/activate
pip install -r requirements.txt
cp ./.env_template ./.env
```

### For .env

for NP_TOKEN
in logined grass web console run
```
localStorage.getItem('np_token');
```


### gerneral usage 

```
source ./.venv/bin/activate
python3 ./nodepay_no_proxy.py
```
