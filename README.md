# nodepay

## usage

### first usage 
```
python3 -m venv ./.venv
source ./.venv/bin/activate
pip install -r requirements.txt
cp ./.env_template ./.env
```

### For .env

for GRASS_USERID and API_KEY
in logined grass web console run
```
localStorage.getItem('accessToken');
localStorage.getItem('userId');
```


### gerneral usage 

```
source ./.venv/bin/activate
python3 ./run.py 
```
